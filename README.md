# QBX Core

Deterministic storage core for persistent AI agent memory:
- Deterministic vault storage
- Snapshot-based versioning
- Cryptographic integrity verification
- Portable deterministic memory containers (Memory Packs)

## Status
- Vault: stable
- Snapshots: working
- Tests: passing
